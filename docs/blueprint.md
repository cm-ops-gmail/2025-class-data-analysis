# **App Name**: SheetDash

## Core Features:

- Data Import: Import data from Google Sheets using its URL. The app will attempt to keep column formatting during the import.
- Automated Field Recognition: Use AI to automatically recognize fields. As a tool, the LLM will decide whether to include data found inside the linked PPTX file.
- Customizable Dashboard View: Allow users to select which columns to display and customize their order in the dashboard view.
- Filtering and Sorting: Enable filtering and sorting of data based on multiple columns.
- Data Editing: Allow users to edit data directly within the dashboard.
- Real-time Updates: Automatically update the dashboard when changes are made in the Google Sheet.
- Search Functionality: Implement a search bar to quickly find specific entries within the data.

## Style Guidelines:

- Primary color: Dark blue (#2C3E50) for a professional and trustworthy feel.
- Background color: Light gray (#F4F4F8) for a clean and modern look.
- Accent color: Teal (#3498DB) for interactive elements and highlights.
- Body and headline font: 'Inter' (sans-serif) for clear readability and a modern, neutral look.
- Use simple, geometric icons for data representation and navigation.
- Use a grid-based layout for the dashboard to ensure consistency and easy navigation.
- Subtle animations for data updates and loading states to improve user experience.
